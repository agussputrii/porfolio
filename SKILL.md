- Display = Archivo Black, ALL CAPS, tight tracking. Body = Inter. Mono = JetBrains Mono, all caps, letter-spaced.
- Corner radius is ZERO. No drop shadows. No gradients. No emoji.
- Signature move: acid lime block highlighting one keyword in a headline.
- Section markers (`§ 02 / MECHANICS`), file tags (`FILE 001 — DRAFT`) on every chrome.
- Voice: declarative, technical, bilingual ES + EN, first-person singular, no SaaS plural.
