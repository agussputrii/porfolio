# Portfolio UI Kit — BRUTAL.DIGITAL

A clickable portfolio prototype for selling digital products & AI services (web dev, chatbots, local AI, audits, CRM/CLM). Built in the brutalist-tech aesthetic of the parent design system.

## Files
- `index.html` — the entry point. Renders all surfaces and lets you click between them.
- `Chrome.jsx` — Topbar + Footer + the persistent file/chrome elements.
- `Hero.jsx` — Home hero with oversized display + acid highlight + CTAs.
- `ProjectIndex.jsx` — Filterable list of case studies.
- `CaseStudy.jsx` — Long-form project detail.
- `About.jsx` — Personal bio surface.
- `Contact.jsx` — Form + direct contact.
- `NotFound.jsx` — Playful 404.

## Surfaces covered
- Home / landing
- Project index
- Case study / project detail
- About
- Contact
- 404
